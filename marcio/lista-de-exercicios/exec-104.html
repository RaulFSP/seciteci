Qual é a diferença entre linguagem de programação Server Side e Client Side?

linguagem Server Side

As linguagens de programação do Server Side são executadas no servidor e enviadas ao cliente, daí o resultado é mostrado ao usuário.O propósito delas são processar os inputs do usuário, validar com banco de dados e gerar tags HTML de maneira dinâmica.

Linguagens Client Side

As linguagens de programação do Client Side são executadas, usualmente, no navegador do usuário. Este navegador web serve de cliente para o servidor. No geral, o propósito é desta vertente são: validações, animações e melhoramentos a experiência do usuário final.
